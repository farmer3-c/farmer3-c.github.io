## 目的

本章中, 将实现一个能处理 `main` 函数和 `return` 语句的编译器. 编译器会将如下的 [SysY](https://gitlab.eduxiji.net/csc1/nscscc/compiler2023/-/blob/master/SysY2022%E8%AF%AD%E8%A8%80%E5%AE%9A%E4%B9%89-V1.pdf) 程序:

```c
int main() {
  // 注释也应该被删掉哦
  return 0;
}
```

编译为对应的 Koopa IR:

```koopa
fun @main(): i32 {
%entry:
  ret 0
}
```
